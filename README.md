# TranscriptsKE

TranscriptsKE is a Kenyan academic transcript request and verification platform.

## Features

- Student registration and login
- Institution selection
- Transcript request workflow
- Admin request review
- Request status tracking
- Future support for M-Pesa, card payments, and document verification

## Tech Stack

- Backend: FastAPI
- Database: PostgreSQL
- ORM: SQLAlchemy
- Migrations: Alembic
- Package manager: uv
- Frontend: Next.js later
